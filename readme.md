# Demon Run
## _Working title_

Demon Run is a game (in the making), for Unity, written in C# and ShaderLab.
Our character is  half demon, half angel. His/her parents tell him that nobody, not in hell or heaven wants him, and that the hunt has started.
Outcasted, our character has to choose to do "demon" or "angel" actions, use "demon" or "angel" weapons, and even its looks will reflect the side that is chosing, as well as his/her skills will be different depending on actions.

- Platform Game in 3D
- Some times is a 3rd person runner.
- ✨Some times is a Fight Game ✨
- Some times you solve puzzles.
## Features

- New property character controller.
- Lava flow designed for hell
- Goo and liquids mechanics, as well as metaballs.
- It works with normal Unity Rendered.
- Original music by the author.
